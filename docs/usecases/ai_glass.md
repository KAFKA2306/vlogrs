# ユースケース：AI Glass（ウェアラブル）の主観体験

メガネ型デバイスを通じて、「見たもの」を事実としてではなく、あなたの「主観的な体験」として再構成するためのガイドラインです。

## 1. 身体と一体化した自然な記録（Seamless Wearable）

デバイスの存在を意識させず、バッテリーや熱を最適化した状態で、生活の断片を捉えます。

- **「決定的な瞬間」の自動ハイライト**:
    - 特定の動作や写真をトリガーに、その前後の脈動を「重要な出来事」として優先的に物語へと昇華させます。

## 2. 視覚と聴覚の完全な同期（Absolute Integration）

「見たもの」と「聞いたこと」を、AIが物理的な一貫性を持って紐付けます。

- **情景の言語化**:
    - AIが視覚情報を分析し、単なる音だけでは伝わらない「光の射し込み」「空気の透明感」といった情景を描写に加えます。
- **物理的な時間同期**:
    - 映像と音声を `±0.05秒` の誤差なく同期させ、世界とのズレを完全に排します。

## 3. 「究極の一人称」による叙述

AIは「何が起きたか」ではなく、「あなたがどう感じたか」を中心に物語を書きます。

- **Show, Don't Tell による主観描写**:
    - 「美しい景色を見た」と書くのではなく、眩しさに細める目、一瞬止まる足取り、高鳴る鼓動などを描写に反映します。
- **Cinematic Aesthetic（記憶の美化）**:
    - `Z-Image-Turbo` を活用し、現実の映像を「記憶の中で輝く美しい一瞬（Golden Hour）」として再構成した挿絵を生成します。

---

## 達成基準（Definition of Checklist）

- [ ] **物理同期の完遂**: 映像フレームと音声が±0.05秒以内の誤差で完全に一致していること。
- [ ] **没入感覚の維持**: 小説内に、デバイスへの言及（カメラ、マイク等）が一切なく、すべてが「私の視界」「私の耳」として記述されていること。
- [ ] **叙事的な美しさ**: 生成された物語が「Standard (3)」を超え、情緒的価値の高い「Excellent (4)」以上の評価を安定して得られること。
