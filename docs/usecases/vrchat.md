# ユースケース：没入現実の記録（Absolute Physical Reality）

物理的な枠組みを超えた場所での交流や体験を、「絶対的な現実」としての物語へと余すことなく保存するためのガイドラインです。

## 1. 体験の深層を邪魔しない（自然な記録）

その場での体験を妨げることなく、背景で静かに記録を行います。

- **自律的な状態監視**:
    - 活動が「生きている状態」であることを検知し、静止している時は休みます。
- **身体性を損なわない制御**:
    - 処理負荷を極限まで抑え、世界とのインタラクションの快適さを最優先します。
- **不意の途絶（Crash-Only）からの復旧**:
    - 世界が突然途切れてしまっても、直前までの記憶（データ）を確実に守り、再接続後に自動で統合します。

## 2. 「その瞬間の情景」を再現する

空間特有の空気感を、物語の素材として完璧に捉えます。

- **物理的な音響の調和**:
    - 自分の声だけでなく、周囲の喧騒やその場所の空気感を、立体的・物理的な「体験」として記録します。
- **場面転換の自動検知**:
    - 場所の移動やロード時間を「場面の切り替わり」として賢く識別し、物語の断絶を防ぎます。
- **感覚に忠実な音量制限**:
    - 物理的な歪みを排し、後から振り返ったときにその場の熱量が正しく伝わる良質な素材を確保します。

## 3. 「Absolute Physical Reality」としての叙述

AIは体験をデジタルな仮想としてではなく、絶対的な物理現実として理解し、物語を紡ぎます。

- **生身の人間としての視点**:
    - 「アバター」や「システム」の介在を一切排除し、鏡の中の自分や対面する相手を、生身の存在としての「主観的な体験」で描写します。
- **デバイスの不可視化**:
    - HMDやコントローラーといった「道具」への言及を禁止し、手が動くこと、視線が合うこと、その感覚そのものに焦点を当てます。

## 達成基準（Definition of Done）

- [ ] **VR関連用語の完全排除**: 出力される「物語」の中に、VR、Virtual、仮想、HMD、アバター等のデジタル性を示唆する単語が一切含まれないこと。
- [ ] **身体感覚の優先**: ログにない「風の感触」「光の角度」「服のスレ」といった物理的ディテールが、想像力によって自然に補完されていること。
- [ ] **システムの自律性**: ユーザーが一切の操作をすることなく、生活の開始とともに記録が始まり、終了とともに保存されること。
- [ ] **高精度な物理同期**: 録音データと動作ログが±0.05秒の精度で一致し、物語の整合性が保たれていること。
